# 🏦 AI-Powered Banking Support & Fraud Intelligence

An intelligent banking assistant designed to improve customer support, fraud detection, and operational decision-making using Artificial Intelligence and Retrieval-Augmented Generation (RAG).

## Features

- 🔍 Intent Classification for banking customer queries
- 😊 Transformer-based Sentiment Analysis
- ⚠️ Explainable Risk Scoring Engine
- 🤖 Automated Action Recommendation Engine
- 📚 RAG-powered Banking Knowledge Retrieval using FAISS
- 🗄️ MySQL integration for support tickets, transactions, and QA pairs
- 📊 Interactive Streamlit Dashboard

## Technologies Used

- Python
- Streamlit
- MySQL
- Scikit-learn
- Hugging Face Transformers
- LangChain
- FAISS Vector Database
- Pandas
- PyTorch

## System Workflow

Customer Query
→ Intent Classification
→ Sentiment Analysis
→ RAG Knowledge Retrieval
→ Risk Assessment
→ Action Recommendation
→ Banking Support Dashboard

## Example Use Cases

- Unauthorized transactions
- Phishing and scam detection
- KYC verification requests
- Loan application support
- Account access issues
- Fraud investigation support
